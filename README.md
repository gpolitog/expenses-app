# Expenses App

A hybrid application build with Quasar Framework to make your own financial control. Edit

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
